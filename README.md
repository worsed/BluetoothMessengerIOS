# Bluetooth Messenger для iOS

Мессенджер для обмена сообщениями между iPhone/iPad через Bluetooth и Wi-Fi Direct.

## 🚀 Быстрая установка

**Есть Mac:**
1. Откройте проект в Xcode
2. Подключите iPhone
3. Нажмите Run (▶️)
4. Готово!

**НЕТ Mac:**
1. Используйте GitHub Actions для сборки (бесплатно!)
2. Скачайте IPA файл
3. Установите через AltStore
4. Готово!

📖 Подробные инструкции:
- **С Mac:** [QUICK_START.md](QUICK_START.md)
- **БЕЗ Mac:** [NO_MAC_GUIDE.md](NO_MAC_GUIDE.md)
- **GitHub Actions:** [GITHUB_ACTIONS_SETUP.md](GITHUB_ACTIONS_SETUP.md)
- **Все способы:** [DEPLOYMENT.md](DEPLOYMENT.md)

## Возможности

- 📱 Автоматическое обнаружение устройств поблизости
- 💬 Обмен текстовыми сообщениями в реальном времени
- 💾 Сохранение истории сообщений (SwiftData)
- 🔔 Push-уведомления о новых сообщениях
- 🔊 Звуковые уведомления и вибрация
- 🔄 Автоматическое переподключение
- 📡 Работает через Bluetooth и Wi-Fi Direct (MultipeerConnectivity)
- 🌙 Поддержка фонового режима
- 🎨 Современный SwiftUI интерфейс

## Требования

- iOS 18.0+
- Xcode 16.0+
- Swift 6.0+
- Устройства с Bluetooth и Wi-Fi

## Технологии

- **SwiftUI** - современный декларативный UI
- **MultipeerConnectivity** - P2P связь через Bluetooth/Wi-Fi
- **SwiftData** - локальное хранилище сообщений
- **UserNotifications** - системные уведомления
- **Combine** - реактивное программирование

## Установка

1. Откройте проект в Xcode:
```bash
open BluetoothMessengerIOS.xcodeproj
```

2. Выберите целевое устройство (iPhone/iPad)

3. Нажмите Run (⌘R)

## Использование

1. Запустите приложение на двух устройствах
2. Устройства автоматически найдут друг друга
3. Начните обмен сообщениями!

### Настройки

- **Автоматическое переподключение** - восстанавливает связь при разрыве
- **Фоновый режим** - принимает сообщения в фоне
- **Звук и вибрация** - уведомления о новых сообщениях

## Как это работает

### MultipeerConnectivity Framework

Приложение использует Apple MultipeerConnectivity, который:
- Автоматически выбирает лучший канал связи (Bluetooth или Wi-Fi Direct)
- Работает без интернета
- Поддерживает до 8 устройств одновременно
- Радиус действия: до 30 метров

### Архитектура

```
ContentView (UI)
    ↓
MessengerViewModel (Логика)
    ↓
MultipeerService (Связь)
    ↓
SwiftData (Хранилище)
```

## Разрешения

Приложение запрашивает:
- **Локальная сеть** - для MultipeerConnectivity
- **Уведомления** - для push-уведомлений

## Примечания

- Устройства должны быть в радиусе действия
- Bluetooth и Wi-Fi должны быть включены
- Не требуется подключение к интернету
- Работает между iPhone, iPad и Mac (с поддержкой MultipeerConnectivity)

## Безопасность

- Все соединения зашифрованы (MCEncryptionRequired)
- Данные передаются только между подключенными устройствами
- История хранится локально на устройстве
