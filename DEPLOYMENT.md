# Как установить приложение на iPhone

Есть несколько способов установки, от самого простого до профессионального.

## 🚀 Способ 1: Прямая установка через Xcode (САМЫЙ ПРОСТОЙ)

### Требования:
- Mac с Xcode
- iPhone с кабелем Lightning/USB-C
- Бесплатный Apple ID (не нужен платный аккаунт разработчика!)

### Шаги:

1. **Подключите iPhone к Mac**
   - Используйте кабель
   - На iPhone нажмите "Доверять этому компьютеру"

2. **Откройте проект в Xcode**
   ```bash
   open BluetoothMessengerIOS.xcodeproj
   ```

3. **Настройте подписание**
   - Выберите проект в навигаторе слева
   - Targets → BluetoothMessengerIOS
   - Signing & Capabilities
   - Team: выберите ваш Apple ID (или добавьте через "Add Account...")
   - Bundle Identifier: измените на уникальный (например, `com.yourname.btmessenger`)

4. **Выберите ваш iPhone**
   - В верхней панели Xcode выберите ваш iPhone из списка устройств

5. **Запустите приложение**
   - Нажмите кнопку Run (▶️) или ⌘R
   - Первый раз может попросить ввести пароль Mac

6. **Доверьте разработчику на iPhone**
   - Настройки → Основные → VPN и управление устройством
   - Найдите ваш Apple ID
   - Нажмите "Доверять"

✅ **Готово!** Приложение установлено и работает 7 дней (потом нужно переустановить)

## 📱 Способ 2: TestFlight (для тестирования с друзьями)

### Требования:
- Платный Apple Developer аккаунт ($99/год)
- Mac с Xcode

### Шаги:

1. **Зарегистрируйтесь в Apple Developer Program**
   - Перейдите на https://developer.apple.com/programs/
   - Оплатите $99/год

2. **Создайте App ID**
   - Войдите в https://developer.apple.com/account
   - Certificates, Identifiers & Profiles
   - Identifiers → + (плюс)
   - App IDs → Continue
   - Bundle ID: `com.yourname.btmessenger`
   - Capabilities: включите "Network Extensions"

3. **Настройте проект в Xcode**
   - Signing & Capabilities
   - Team: выберите вашу платную команду
   - Automatically manage signing: включено

4. **Создайте приложение в App Store Connect**
   - Перейдите на https://appstoreconnect.apple.com
   - My Apps → + (плюс) → New App
   - Platforms: iOS
   - Name: Bluetooth Messenger
   - Bundle ID: выберите созданный ранее
   - SKU: btmessenger001

5. **Архивируйте приложение**
   - В Xcode: Product → Destination → Any iOS Device
   - Product → Archive
   - Дождитесь завершения

6. **Загрузите в TestFlight**
   - Откроется Organizer
   - Выберите архив → Distribute App
   - TestFlight & App Store → Next
   - Upload → Next → Upload
   - Дождитесь обработки (10-30 минут)

7. **Добавьте тестировщиков**
   - App Store Connect → TestFlight
   - Internal Testing или External Testing
   - Добавьте email тестировщиков
   - Они получат приглашение

8. **Установка на iPhone**
   - Установите TestFlight из App Store
   - Откройте ссылку из приглашения
   - Нажмите "Установить"

✅ **Преимущества:** 
- Работает 90 дней
- До 10,000 тестировщиков
- Автоматические обновления

## 🔧 Способ 4: Sideload через AltStore (без Mac)

### Требования:
- Windows/Mac/Linux компьютер
- AltStore/AltServer
- Бесплатный Apple ID

### Шаги:

1. **Установите AltServer**
   - Скачайте с https://altstore.io
   - Установите на компьютер

2. **Установите AltStore на iPhone**
   - Подключите iPhone к компьютеру
   - Запустите AltServer
   - Install AltStore → выберите iPhone
   - Введите Apple ID и пароль

3. **Создайте IPA файл**
   - В Xcode: Product → Archive
   - Distribute App → Custom → Next
   - App Thinning: None
   - Rebuild from Bitcode: No
   - Export → выберите папку
   - Получите файл `.ipa`

4. **Установите через AltStore**
   - Откройте AltStore на iPhone
   - My Apps → + (плюс)
   - Выберите `.ipa` файл
   - Дождитесь установки

✅ **Преимущества:**
- Не нужен Mac для переустановки
- Работает 7 дней, но можно обновлять через Wi-Fi

## 📦 Способ 5: App Store (публичный релиз)

### Требования:
- Платный Apple Developer аккаунт ($99/год)
- Прохождение App Review

### Шаги:

1. Выполните шаги 1-5 из "Способ 2: TestFlight"

2. **Подготовьте метаданные**
   - App Store Connect → My Apps → ваше приложение
   - Заполните:
     - Описание
     - Скриншоты (обязательно!)
     - Ключевые слова
     - Категория: Social Networking
     - Возрастной рейтинг

3. **Создайте скриншоты**
   - Запустите приложение в симуляторе
   - ⌘S для скриншота
   - Нужны размеры: 6.7", 6.5", 5.5"

4. **Отправьте на ревью**
   - App Store Connect → Submit for Review
   - Ответьте на вопросы
   - Submit

5. **Ожидайте проверки**
   - Обычно 1-3 дня
   - Могут запросить дополнительную информацию

✅ **После одобрения:** приложение доступно всем в App Store!

## ⚠️ Важные замечания

### Бесплатный Apple ID (Способ 1):
- ✅ Бесплатно
- ❌ Приложение работает только 7 дней
- ❌ Максимум 3 приложения одновременно
- ❌ Нужно переустанавливать каждую неделю

### Платный аккаунт ($99/год):
- ✅ Приложения работают 1 год
- ✅ TestFlight доступен
- ✅ Можно публиковать в App Store
- ✅ Неограниченное количество приложений

### Разрешения:
Приложение запросит:
- **Локальная сеть** - обязательно разрешите!
- **Уведомления** - для получения сообщений

### Тестирование:
- Симулятор НЕ поддерживает MultipeerConnectivity
- Нужны 2 реальных iPhone/iPad для тестирования
- Устройства должны быть в радиусе 30 метров

## 🆘 Решение проблем

### "Untrusted Developer"
→ Настройки → Основные → VPN и управление устройством → Доверять

### "Unable to install"
→ Проверьте Bundle Identifier (должен быть уникальным)

### "Signing certificate expired"
→ Переустановите приложение через Xcode

### Приложение не находит устройства
→ Разрешите доступ к локальной сети в настройках iOS
